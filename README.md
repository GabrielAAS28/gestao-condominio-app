# App Gestão de Condomínio

Aplicação mobile desenvolvida com React Native para gerenciamento de condomínios.

## Tecnologias

- React Native
- JavaScript
- REST API
- Redux

## Funcionalidades

- Cadastro de moradores
- Gestão de reservas
- Listagem de avisos
- Interface mobile responsiva
